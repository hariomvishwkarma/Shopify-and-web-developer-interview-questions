4. What is the difference between Liquid and JSON in creating templates?
Liquid is a templating language that allows for dynamic content rendering on the server side, while JSON is a data format used for data interchange. Liquid can integrate data from JSON to display it within templates.
5. How do you add functionality to a specific product?
To add functionality to a specific product, you can use product tags, metafields, or conditional statements in Liquid to customize the display and functionality based on the product's attributes.
6. What is the difference between Metafield and Metaobject in Shopify?
Metafields are used to store additional data for individual resources (like products, collections, customers).
Metaobjects are reusable custom data models that can store structured data and be linked to multiple resources.

Example:
- Metafield: A single product’s “ingredient list”
- Metaobject: A shared “ingredient” data model used across multiple products.
7. Define Filter in Liquid.
Filters in Liquid are used to modify the output of variables. They are similar to functions that transform data.
Example:
{{ product.title | upcase }} → converts product title to uppercase.
8. What is App Development in Shopify?
App development in Shopify involves creating private, custom, or public apps to extend store functionality using the Shopify API, Admin API, and Storefront API. Apps can handle automation, integrate external systems, or add custom storefront features.
9. Define Schema in Shopify.
Schema defines the settings and structure of a section in a Shopify theme. It allows you to add editable fields (like text, image, color pickers) in the theme editor.
Example:
      {% schema %}
      {
        "name": "Hero Banner",
        "settings": [
          {
            "type": "text",
            "id": "heading",
            "label": "Banner Heading"
          }
        ]
      }
      {% endschema %}
10. What is the difference between Assign and Capture in Liquid?
assign is used to store a simple value in a variable.
capture is used to store blocks of text or HTML in a variable.

Example:
      {% assign name = 'Harion' %}
      {% capture message %}
        Hello {{ name }}, welcome to Shopify!
      {% endcapture %}
      {{ message }}
