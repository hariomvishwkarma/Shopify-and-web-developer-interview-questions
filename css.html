1. What are pseudo-elements and pseudo-classes in CSS?
Pseudo-elements allow you to style specific parts of an element, such as `::before` or `::after`, while pseudo-classes, such as `:hover`, define a special state of an element.
2. How do you center text inside a div?
To center text horizontally, you can use `text-align: center;`. For vertical centering, you can use `display: flex;` along with `align-items: center;` and `justify-content: center;`.
3. What is the box model in CSS?
The box model consists of content, padding, border, and margin. Each of these layers affects the total width and height of the element.
4. How do you create a responsive layout using media queries?
Media queries allow you to apply different styles based on the screen size or device type. For example: 
        @media (max-width: 768px) {
          .container {
            width: 100%;
          }
        }
5. What is Flexbox and how does it work?
Flexbox is a layout model that allows items to align and distribute space within a container. It provides flexibility in creating responsive layouts. Use `display: flex;` on the parent element and `flex` properties on child elements to control alignment.
