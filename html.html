1. What is the difference between <div> and <section>?
The <div> element is a generic container for flow content, which does not carry any specific semantic meaning. The <section> element is used to define sections in a document, such as chapters, headers, footers, or any other sections of the content that represent a thematic grouping of content.
2. What is semantic HTML and why is it important?
Semantic HTML refers to the use of HTML elements to reinforce the meaning of the information in webpages, rather than merely defining its presentation. It is important for accessibility, SEO, and providing meaningful context to browsers and other tools.
