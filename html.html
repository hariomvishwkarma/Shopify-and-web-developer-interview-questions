1. What is the difference between <div> and <section>?
The <div> element is a generic container for flow content, which does not carry any specific semantic meaning. The <section> element is used to define sections in a document, such as chapters, headers, footers, or any other sections of the content that represent a thematic grouping of content.
2. What is semantic HTML and why is it important?
Semantic HTML refers to the use of HTML elements to reinforce the meaning of the information in webpages, rather than merely defining its presentation. It is important for accessibility, SEO, and providing meaningful context to browsers and other tools.
3. How do you create a form in HTML? Explain with examples.
A form in HTML can be created using the <form> element, which contains <input>, <textarea>, and <button> elements. For example: 
        <form action="/submit" method="POST">
          <input type="text" name="username" placeholder="Username">
          <input type="password" name="password" placeholder="Password">
          <button type="submit">Submit</button>
        </form>
4. What are the different types of input elements in HTML forms?
Input elements in HTML forms include text, password, radio, checkbox, file, submit, button, date, email, and more. Each input type allows for different kinds of user input.
5. How does the <meta> tag work in HTML? Provide examples.
The <meta> tag provides metadata about the HTML document, such as author, description, and keywords, as well as control over browser behavior like viewport settings. Example: 
        <meta name="description" content="Interview questions for web developers">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
