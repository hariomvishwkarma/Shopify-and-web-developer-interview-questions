1. What is setInterval and how do you use it?
`setInterval` is a method that repeatedly calls a function with a fixed time delay between each call. For example: 
        setInterval(() => {
          console.log('This will run every second');
        }, 1000);
2. What is setTimeout and how do you use it?
`setTimeout` is a method that executes a function after a specified number of milliseconds. For example: 
        setTimeout(() => {
          console.log('This will run after 2 seconds');
        }, 2000);
3. What are closures in JavaScript?
Closures are functions that retain access to their outer scope, even when the outer function has finished executing. They allow for private variables.
4. What is the difference between `==` and `===`?
`==` compares values for equality after performing type coercion, while `===` checks for both value and type equality without coercion.
5. How do you handle asynchronous operations in JavaScript?
You can handle asynchronous operations using callbacks, promises, or the `async/await` syntax. For example: 
        async function fetchData() {
          const response = await fetch('https://api.example.com/data');
          const data = await response.json();
          console.log(data);
        }
